Can You Learn Web Design On Your Own?

If you want to learn web design on your own, we have good news: it is possible for you to gain the knowledge and skills required in designing websites, over time. In the strict sense of the word, you can’t really learn web design “on your own” since you will be relying on the knowledge of experienced web designers who are generous enough to share their expertise with self-learners like you.

What have you to learn?

If you would like to be a web devolper you have to know about Hypertext Markup Language (HTML) and Cascading Style Sheets (CSS). You can read more about them.

youtube

Our channel Traversy-Media on youtube can give you useful tips and tricks in web design, help you avoid common web design mistakes, and teach you the latest and best practices in designing websites.

Learn HTML

In this crash course I will cram as much about HTML that I can. This is meant for absolute beginners. If you are interested in learning HTML but know nothing, then you are in the right place. We will be creating a cheat sheet with all of the common HTML5 tags, attributes, semantic markup, etc. We will not be focusing on CSS in this video. The CSS crash course will be released shortly after

Learn CSS

In this video I will cram as much as possible about CSS. We will be looking at styles, selectors, declarations, etc. We will build a CSS cheat sheet that you can keep as a resource and we will also create a basic website layout. We are using CSS3 but mostly the basics. I will be creating an advanced CSS course with animations, etc. I do have a Flexbox in 20 minutes video as well if you want to learn flexbox.

Learn Js

In this crash course we will go over the fundamentals of JavaScript including more modern syntax like classes, arrow functions, etc. This is the starting point on my channel for learning JS.

blog

If you want to learn web design without spending a dime, we recommend that you start reading blogs of web designers online. Our blog can give you useful tips and tricks in web design, help you avoid common web design mistakes, and teach you the latest and best practices in designing websites.
